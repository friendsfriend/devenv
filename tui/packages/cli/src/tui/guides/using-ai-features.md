# Using AI Features

DevEnv integrates with pi for code review, log analysis, and interactive sessions.

## 1. Pi session view

Press `A` in the table view to open the pi session view. This shows a new-session entry and recent pi sessions grouped by project.

The pi session view has two sections:

- **New Session** — Start a fresh pi session
- **Recent Sessions** — Resume a previous pi session (sorted by last updated)

## 2. Starting a new session

1. Press `A` to open the pi session view
2. Make sure **New Session** is selected (top of the list)
3. Press `Enter` to launch a new session

This opens the pi agent in a new terminal session.

## 3. Resuming a session

1. Press `A` to open the pi session view
2. Navigate to the session you want to resume using `j`/`k`
3. Press `Enter` to resume that session

Sessions are sorted by last updated time, most recent first.

## 4. Searching sessions

Press `/` while in the pi session view to filter sessions by title or project. Type to narrow down, press `Enter` to select, or `Esc` to clear the filter.

## 5. AI in change request detail

While viewing a change request detail, press `Shift+A` to trigger a pi-powered AI review:

1. The review is generated by pi and streamed into an overlay
2. Review content can be posted as an CR comment
3. Use `Ctrl+j`/`Ctrl+k` to scroll the review overlay

## 6. AI in log viewer

While viewing logs, press `Shift+A` to trigger AI analysis on the current log content:

1. In normal mode: enters prompt mode — type what you want to analyze
2. In visual mode: selects the highlighted log range for analysis
3. Results are shown in an overlay with scroll support
4. Type follow-up questions to refine the analysis

## 7. pi integration

DevEnv integrates with the pi coding agent for interactive development workflows. Sessions are managed through the pi session view and can be started or resumed directly from the TUI.

See [Using Git Integrations](using-git-integrations.md) for AI-powered code review features.
